# ZAJEL Delivery Platform — Professional Version

A polished logistics portfolio project for a UAE delivery company.

## Features
- Professional landing page
- Responsive design
- Dark mode
- Shipment tracking demo
- Smart delivery quote calculator
- UAE coverage section
- Customer dashboard
- Admin dashboard
- Login / registration demo using LocalStorage
- Express server with shipment API demo

## Run locally
```bash
npm install
npm start
```
Then open:
```text
http://localhost:3000
```
